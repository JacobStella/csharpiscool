## What code is ##
1. Code is a set of instructions a computer follows.

2. Computers do exactly what you tell them, not what you meant.

3. Programs are built from small, simple steps combined logically.

4. Every language (Python, JavaScript, C++, etc.) has different syntax but similar core concepts.
